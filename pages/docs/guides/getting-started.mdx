# Getting Started

Welcome to ZBR! This guide will help you set up your development environment and create your first Discord bot.

## Prerequisites

- Node.js 18 or higher
- A Discord Bot Token (from the [Discord Developer Portal](https://discord.com/developers/applications))

## Installation

Install the ZBR CLI globally using npm:

```bash
npm install -g @zbrlang/zbr
```

## Creating a Project

1. Initialize a new ZBR project:
   ```bash
   zbr init my-awesome-bot
   ```
2. Navigate to your project directory:
   ```bash
   cd my-awesome-bot
   ```
3. Open `zbr.json` and paste your bot token:
   ```json
   {
     "name": "my-awesome-bot",
     "token": "YOUR_TOKEN_HERE",
     "prefix": "!"
   }
   ```

## Running the Bot

Start the bot in development mode with hot-reloading enabled:

```bash
zbr dev
```

Your bot is now online! Try typing `!ping` in your Discord server.
