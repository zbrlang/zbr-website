# Extensions

Official IDE extensions for ZBR.

There are currently two supported extensions:

- **VS Code**: ready for use, with syntax highlighting, snippets, smart completions, and function references.
- **Zed**: currently in development with Tree-sitter-powered language support and native editor integration.

## What’s included

- Full ZBR syntax highlighting in supported editors
- Snippets for common ZBR patterns and functions
- Smart completions for available ZBR functions and triggers
- Quick links to the extension repositories and marketplace pages

## Where to find them

See the extensions overview page in the app at `/extensions` for the latest status, installation links, and repository details.
