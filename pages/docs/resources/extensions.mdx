# Extensions

Official IDE extensions for ZBR.

There are currently two supported extensions:

- **VS Code**: Official VS Code extension providing syntax highlighting and 400+ snippets.
- **Zed**: Official Zed extension powered by a Tree-sitter grammar.

## What’s included

- Full ZBR syntax highlighting in supported editors
- Snippets for common ZBR patterns and functions
- Smart completions for available ZBR functions and triggers
- Quick links to the extension repositories and marketplace pages

## Where to find them

See the extensions overview page in the app at `/extensions` or [here](https://zbr-website.vercel.app/extensions) for the latest status, installation links, and repository details.
