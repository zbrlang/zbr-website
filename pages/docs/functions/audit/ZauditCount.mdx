# ZauditCount

Fetches the guild audit log and returns the number of returned entries.

## Syntax

```zbr
ZauditCount{guildID?;limit?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |
| limit | integer | No | 20 |



## Example

```zbr
ZauditCount{}
```
