# ZauditEntryID

Returns the audit log entry ID for the selected entry.

## Syntax

```zbr
ZauditEntryID{guildID?;entrySelector?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |
| entrySelector | string | No | most recent entry |



## Example

```zbr
ZauditEntryID{}
```
