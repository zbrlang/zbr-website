# ZauditLatest

Fetches the most recent guild audit log entry and returns it as a JSON object.

## Syntax

```zbr
ZauditLatest{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZauditLatest{}
```
