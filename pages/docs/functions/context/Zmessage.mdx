# Zmessage

Returns the full message content after the command trigger, or the Nth space-separated word when an index is provided.

## Syntax

```zbr
Zmessage{index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | No | full content |



## Example

```zbr
Zmessage{1}
```
