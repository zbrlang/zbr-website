# Zdiv

Divides the first value by each subsequent divisor in sequence and returns the result.

## Syntax

```zbr
Zdiv{value;divisor;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| value | number | Yes | - |
| divisor | number | Yes | - |



## Example

```zbr
Zdiv{100;5;2}
```
