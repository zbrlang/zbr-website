# Zround

Rounds a number to the specified number of decimal places, defaulting to zero decimal places.

## Syntax

```zbr
Zround{value;decimals?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| value | number | Yes | - |
| decimals | integer | No | 0 |



## Example

```zbr
Zround{3.14159;2}
```
