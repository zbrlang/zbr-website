# Zmodulo

Returns the remainder of dividing the dividend by the divisor.

## Syntax

```zbr
Zmodulo{dividend;divisor}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| dividend | number | Yes | - |
| divisor | number | Yes | - |



## Example

```zbr
Zmodulo{10;3}
```
