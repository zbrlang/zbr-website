# Zcalculate

Evaluates a mathematical expression supporting +, -, *, /, %, and ** operators with parentheses and correct operator precedence.

## Syntax

```zbr
Zcalculate{expression}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| expression | string | Yes | - |



## Example

```zbr
Zcalculate{(2 + 3) * 4 ** 2}
```
