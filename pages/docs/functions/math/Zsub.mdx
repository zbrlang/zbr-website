# Zsub

Subtracts each subsequent number from the first value and returns the result.

## Syntax

```zbr
Zsub{value;number;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| value | number | Yes | - |
| number | number | Yes | - |



## Example

```zbr
Zsub{10;3;2}
```
