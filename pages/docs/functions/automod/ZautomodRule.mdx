# ZautomodRule

Returns information about a specific automod rule in the guild as JSON.

## Syntax

```zbr
ZautomodRule{ruleID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| ruleID | snowflake | Yes | - |



## Example

```zbr
ZautomodRule{123456789012345678}
```
