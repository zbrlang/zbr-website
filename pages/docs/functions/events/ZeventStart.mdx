# ZeventStart

Returns the start time of a scheduled event as a Unix timestamp string.

## Syntax

```zbr
ZeventStart{eventID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| eventID | snowflake | Yes | - |



## Example

```zbr
ZeventStart{1234567890123456789}
```
