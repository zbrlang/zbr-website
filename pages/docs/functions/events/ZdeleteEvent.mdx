# ZdeleteEvent

Deletes a scheduled event from the current server by its event ID.

## Syntax

```zbr
ZdeleteEvent{eventID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| eventID | snowflake | Yes | - |



## Example

```zbr
ZdeleteEvent{1234567890123456789}
```
