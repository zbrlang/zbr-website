# ZeventSubscribers

Returns the number of users subscribed to a scheduled event.

## Syntax

```zbr
ZeventSubscribers{eventID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| eventID | snowflake | Yes | - |



## Example

```zbr
ZeventSubscribers{1234567890123456789}
```
