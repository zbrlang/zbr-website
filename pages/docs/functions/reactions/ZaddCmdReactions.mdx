# ZaddCmdReactions

Immediately adds one or more emoji reactions to the message that triggered the command.

## Syntax

```zbr
ZaddCmdReactions{emoji;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| emoji | emoji | Yes | - |



## Example

```zbr
ZaddCmdReactions{✅;🚀}
```
