# ZaddReactions

Queues one or more emoji reactions to be added to the bot's own response message after it is sent.

## Syntax

```zbr
ZaddReactions{emoji;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| emoji | emoji | Yes | - |



## Example

```zbr
ZaddReactions{👍;❤️;🎉}
```
