# ZemojiExists

Returns true if a custom emoji with the given ID exists in the current server, or false otherwise.

## Syntax

```zbr
ZemojiExists{id}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| id | snowflake | Yes | - |



## Example

```zbr
ZemojiExists{123456789012345678}
```
