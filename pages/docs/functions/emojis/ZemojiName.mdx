# ZemojiName

Returns the name of a custom emoji in the current server looked up by its ID.

## Syntax

```zbr
ZemojiName{id}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| id | snowflake | Yes | - |



## Example

```zbr
ZemojiName{123456789012345678}
```
