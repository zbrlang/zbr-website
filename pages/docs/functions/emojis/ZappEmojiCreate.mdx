# ZappEmojiCreate

Creates a new application emoji from an image URL. Returns the emoji ID.

## Syntax

```zbr
ZappEmojiCreate{name;imageURL}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |
| imageURL | url | Yes | - |



## Example

```zbr
ZappEmojiCreate{wave;https://example.com/wave.png}
```
