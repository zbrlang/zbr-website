# ZisEmojiAnimated

Returns true if the custom emoji with the given ID is animated, or false if it is static.

## Syntax

```zbr
ZisEmojiAnimated{id}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| id | snowflake | Yes | - |



## Example

```zbr
ZisEmojiAnimated{123456789012345678}
```
