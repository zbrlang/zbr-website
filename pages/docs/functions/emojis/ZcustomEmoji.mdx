# ZcustomEmoji

Returns the usable Discord emoji string for a server custom emoji looked up by name.

## Syntax

```zbr
ZcustomEmoji{name}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |



## Example

```zbr
ZcustomEmoji{wave}
```
