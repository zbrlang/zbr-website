# ZroleExists

Returns true if the given role ID exists in the guild, defaulting to the author's top role if no ID is provided.

## Syntax

```zbr
ZroleExists{roleID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| roleID | snowflake | No | author's top role |



## Example

```zbr
ZroleExists{987654321012345678}
```
