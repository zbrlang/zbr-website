# ZroleColor

Returns the hex color code of a role, defaulting to the author's top role if no ID is provided.

## Syntax

```zbr
ZroleColor{roleID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| roleID | snowflake | No | author's top role |



## Example

```zbr
ZroleColor{987654321012345678}
```
