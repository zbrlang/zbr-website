# ZserverRoles

Returns a separator-joined list of role IDs for the specified server.

## Syntax

```zbr
ZserverRoles{separator?;guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | 
 |
| guildID | snowflake | No | - |



## Example

```zbr
ZserverRoles{;123456789012345678}
```
