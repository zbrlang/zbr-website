# ZrolePosition

Returns the hierarchy position of a role, defaulting to the author's top role if no ID is provided.

## Syntax

```zbr
ZrolePosition{roleID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| roleID | snowflake | No | author's top role |



## Example

```zbr
ZrolePosition{987654321012345678}
```
