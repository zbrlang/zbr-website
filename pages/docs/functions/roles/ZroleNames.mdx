# ZroleNames

Returns all role names in the guild sorted by position from highest to lowest, joined by the given separator.

## Syntax

```zbr
ZroleNames{separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | 
 |



## Example

```zbr
ZroleNames{, }
```
