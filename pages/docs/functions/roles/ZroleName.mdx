# ZroleName

Returns the name of a role by its ID, defaulting to the author's top role if no ID is provided.

## Syntax

```zbr
ZroleName{roleID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| roleID | snowflake | No | author's top role |



## Example

```zbr
ZroleName{987654321012345678}
```
