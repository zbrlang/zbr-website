# ZcolorRole

Sets the color of an existing role to the specified hex color code.

## Syntax

```zbr
ZcolorRole{roleID;color}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| roleID | snowflake | Yes | - |
| color | string | Yes | - |



## Example

```zbr
ZcolorRole{987654321012345678;#3498DB}
```
