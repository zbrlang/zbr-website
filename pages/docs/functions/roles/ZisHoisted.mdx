# ZisHoisted

Returns true if the given role is displayed separately in the member list, defaulting to the author's top role if no ID is provided.

## Syntax

```zbr
ZisHoisted{roleID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| roleID | snowflake | No | author's top role |



## Example

```zbr
ZisHoisted{987654321012345678}
```
