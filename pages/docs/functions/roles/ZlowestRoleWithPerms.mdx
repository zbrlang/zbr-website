# ZlowestRoleWithPerms

Returns the ID of the lowest-position role in the guild that has all of the specified permissions.

## Syntax

```zbr
ZlowestRoleWithPerms{permission;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| permission | permission | Yes | - |



## Example

```zbr
ZlowestRoleWithPerms{manageMessages}
```
