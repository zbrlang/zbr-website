# ZwelcomeScreen

Returns the guild's welcome screen as a JSON object.

## Syntax

```zbr
ZwelcomeScreen{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZwelcomeScreen{123456789012345678}
```
