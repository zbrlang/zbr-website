# ZserverIcon

Returns the icon URL of the guild, or an empty string if no icon is set.

## Syntax

```zbr
ZserverIcon{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZserverIcon{}
```
