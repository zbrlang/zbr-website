# ZguildExists

Returns true if a guild with the given ID is accessible by the bot, or false otherwise.

## Syntax

```zbr
ZguildExists{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZguildExists{1234567890123456789}
```
