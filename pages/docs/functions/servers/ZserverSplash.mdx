# ZserverSplash

Returns the splash URL of the guild, or an empty string if no splash is set.

## Syntax

```zbr
ZserverSplash{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZserverSplash{}
```
