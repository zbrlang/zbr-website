# ZguildBanner

Returns the banner URL of the guild, or an empty string if no banner is set.

## Syntax

```zbr
ZguildBanner{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZguildBanner{}
```
