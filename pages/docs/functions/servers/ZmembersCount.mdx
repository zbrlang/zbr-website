# ZmembersCount

Returns the total number of members in the guild, or the count filtered by presence status.

## Syntax

```zbr
ZmembersCount{status?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| status | string | No | - |



## Example

```zbr
ZmembersCount{online}
```
