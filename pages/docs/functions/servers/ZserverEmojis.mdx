# ZserverEmojis

Returns the custom emojis of the guild as formatted emoji strings joined by the specified separator.

## Syntax

```zbr
ZserverEmojis{guildID?;separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |
| separator | string | No | 
 |



## Example

```zbr
ZserverEmojis{;, }
```
