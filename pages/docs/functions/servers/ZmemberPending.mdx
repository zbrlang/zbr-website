# ZmemberPending

Returns true or false whether a guild member has not yet completed membership screening.

## Syntax

```zbr
ZmemberPending{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZmemberPending{123456789012345678}
```
