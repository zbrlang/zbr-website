# ZserverDescription

Returns the description of the guild, or an empty string if none is set.

## Syntax

```zbr
ZserverDescription{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZserverDescription{}
```
