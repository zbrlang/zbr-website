# ZserverDiscoverySplash

Returns the discovery splash URL of the guild, or an empty string if none is set.

## Syntax

```zbr
ZserverDiscoverySplash{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZserverDiscoverySplash{}
```
