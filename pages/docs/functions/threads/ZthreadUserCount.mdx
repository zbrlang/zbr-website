# ZthreadUserCount

Returns the total number of members currently in the specified thread.

## Syntax

```zbr
ZthreadUserCount{threadID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| threadID | snowflake | Yes | - |



## Example

```zbr
ZthreadUserCount{123456789012345678}
```
