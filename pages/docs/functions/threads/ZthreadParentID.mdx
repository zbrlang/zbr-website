# ZthreadParentID

Returns the parent channel ID of the thread, or empty string.

## Syntax

```zbr
ZthreadParentID{threadID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| threadID | snowflake | No | - |



## Example

```zbr
ZthreadParentID{123456789012345678}
```
