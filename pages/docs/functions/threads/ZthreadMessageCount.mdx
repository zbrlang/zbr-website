# ZthreadMessageCount

Returns the number of messages in the specified thread.

## Syntax

```zbr
ZthreadMessageCount{threadID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| threadID | snowflake | Yes | - |



## Example

```zbr
ZthreadMessageCount{123456789012345678}
```
