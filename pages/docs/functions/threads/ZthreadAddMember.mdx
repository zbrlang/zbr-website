# ZthreadAddMember

Adds a user to the specified thread by their user ID.

## Syntax

```zbr
ZthreadAddMember{threadID;userID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| threadID | snowflake | Yes | - |
| userID | snowflake | Yes | - |



## Example

```zbr
ZthreadAddMember{123456789012345678;987654321098765432}
```
