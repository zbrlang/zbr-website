# ZthreadMetadata

Returns JSON metadata for a thread including owner, creation date, and flags.

## Syntax

```zbr
ZthreadMetadata{threadID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| threadID | snowflake | Yes | - |



## Example

```zbr
ZthreadMetadata{123456789012345678}
```
