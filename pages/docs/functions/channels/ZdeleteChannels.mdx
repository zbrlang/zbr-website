# ZdeleteChannels

Deletes one or more channels by their IDs.

## Syntax

```zbr
ZdeleteChannels{channelID;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZdeleteChannels{1234567890123456789;9876543210987654321}
```
