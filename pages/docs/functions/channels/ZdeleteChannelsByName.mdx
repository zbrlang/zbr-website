# ZdeleteChannelsByName

Deletes all channels in the current guild whose names match any of the provided names.

## Syntax

```zbr
ZdeleteChannelsByName{name;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |



## Example

```zbr
ZdeleteChannelsByName{general;off-topic}
```
