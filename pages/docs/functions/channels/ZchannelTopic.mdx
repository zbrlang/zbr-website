# ZchannelTopic

Returns the topic of a Discord text channel, or an empty string if none is set.

## Syntax

```zbr
ZchannelTopic{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZchannelTopic{1234567890123456789}
```
