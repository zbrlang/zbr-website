# ZchannelPosition

Returns the position index of a channel within the guild's channel list.

## Syntax

```zbr
ZchannelPosition{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZchannelPosition{1234567890123456789}
```
