# ZparentID

Returns the ID of the parent category of the given channel, or an empty string if the channel has no parent.

## Syntax

```zbr
ZparentID{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZparentID{1234567890123456789}
```
