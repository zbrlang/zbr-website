# ZchannelNames

Returns a sorted list of all channel names in the guild joined by the specified separator.

## Syntax

```zbr
ZchannelNames{separator?;guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | 
 |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZchannelNames{, }
```
