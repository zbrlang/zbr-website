# ZisNSFW

Returns true if the given channel is marked as NSFW, or false otherwise.

## Syntax

```zbr
ZisNSFW{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZisNSFW{1234567890123456789}
```
