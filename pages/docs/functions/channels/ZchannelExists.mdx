# ZchannelExists

Returns true if a channel with the given ID exists, or false otherwise.

## Syntax

```zbr
ZchannelExists{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZchannelExists{1234567890123456789}
```
