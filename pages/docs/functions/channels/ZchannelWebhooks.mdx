# ZchannelWebhooks

Returns a space-separated list of webhook URLs configured for the given channel.

## Syntax

```zbr
ZchannelWebhooks{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZchannelWebhooks{1234567890123456789}
```
