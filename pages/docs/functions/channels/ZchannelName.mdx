# ZchannelName

Returns the name of a Discord channel by its ID, falling back to the current channel if no ID is provided.

## Syntax

```zbr
ZchannelName{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZchannelName{1234567890123456789}
```
