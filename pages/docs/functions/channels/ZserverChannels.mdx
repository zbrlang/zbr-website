# ZserverChannels

Returns a separator-joined list of channel IDs for the specified server.

## Syntax

```zbr
ZserverChannels{separator?;guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | 
 |
| guildID | snowflake | No | - |



## Example

```zbr
ZserverChannels{;123456789012345678}
```
