# ZrulesChannelID

Returns the ID of the rules channel in the guild, or an empty string if none is configured.

## Syntax

```zbr
ZrulesChannelID{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZrulesChannelID{}
```
