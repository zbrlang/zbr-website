# Zcolor

Sets the accent color of the specified embed slot using a hex color code.

## Syntax

```zbr
Zcolor{hex;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| hex | string | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
Zcolor{#FF5733;1}
```
