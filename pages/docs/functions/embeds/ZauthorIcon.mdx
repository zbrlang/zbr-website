# ZauthorIcon

Sets the author icon image URL on the specified embed slot, requiring an author to already be set.

## Syntax

```zbr
ZauthorIcon{url;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
ZauthorIcon{https://example.com/avatar.png;1}
```
