# ZtitleURL

Sets a hyperlink URL on the embed's title, requiring a title to already be set on that slot.

## Syntax

```zbr
ZtitleURL{url;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
ZtitleURL{https://example.com;1}
```
