# Zimage

Sets the large image URL on the specified embed slot.

## Syntax

```zbr
Zimage{url;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
Zimage{https://example.com/banner.png;1}
```
