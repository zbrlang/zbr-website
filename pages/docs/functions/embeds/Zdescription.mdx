# Zdescription

Sets the description text on the specified embed slot.

## Syntax

```zbr
Zdescription{text;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
Zdescription{This is the embed body.;1}
```
