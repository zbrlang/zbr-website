# ZfooterIcon

Sets the footer icon image URL on the specified embed slot, requiring a footer to already be set.

## Syntax

```zbr
ZfooterIcon{url;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
ZfooterIcon{https://example.com/icon.png;1}
```
