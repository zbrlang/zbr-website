# ZauthorURL

Sets a hyperlink URL on the embed's author name, requiring an author to already be set.

## Syntax

```zbr
ZauthorURL{url;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
ZauthorURL{https://example.com/profile;1}
```
