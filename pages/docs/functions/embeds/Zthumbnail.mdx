# Zthumbnail

Sets the thumbnail image URL on the specified embed slot.

## Syntax

```zbr
Zthumbnail{url;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| index | integer | No | 1 |



## Example

```zbr
Zthumbnail{https://example.com/thumb.png;1}
```
