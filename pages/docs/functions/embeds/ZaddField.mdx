# ZaddField

Adds a named field with a value to the specified embed slot, with an optional inline layout flag.

## Syntax

```zbr
ZaddField{name;value;inline?;index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |
| value | string | Yes | - |
| inline | boolean | No | false |
| index | integer | No | 1 |



## Example

```zbr
ZaddField{Score;9001;true;1}
```
