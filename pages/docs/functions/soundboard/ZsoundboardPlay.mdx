# ZsoundboardPlay

Plays a soundboard sound in a voice channel.

## Syntax

```zbr
ZsoundboardPlay{soundId;voiceChannelId}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| soundId | snowflake | Yes | - |
| voiceChannelId | snowflake | Yes | - |



## Example

```zbr
ZsoundboardPlay{123456789012345678;987654321098765432}
```
