# ZsoundboardCreate

Uploads a new sound to the server soundboard from a file URL.

## Syntax

```zbr
ZsoundboardCreate{name;fileUrl}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |
| fileUrl | url | Yes | - |



## Example

```zbr
ZsoundboardCreate{funny;https://example.com/sound.mp3}
```
