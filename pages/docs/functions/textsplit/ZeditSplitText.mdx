# ZeditSplitText

Replaces the element at the given 1-based index in the current split text with a new value.

## Syntax

```zbr
ZeditSplitText{index;value?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | Yes | - |
| value | string | No | - |



## Example

```zbr
ZeditSplitText{2;pear}
```
