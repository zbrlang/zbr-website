# ZremoveSplitTextElement

Removes the element at the given 1-based index from the current split text.

## Syntax

```zbr
ZremoveSplitTextElement{index}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | Yes | - |



## Example

```zbr
ZremoveSplitTextElement{2}
```
