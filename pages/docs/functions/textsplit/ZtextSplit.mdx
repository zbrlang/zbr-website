# ZtextSplit

Splits a string by a separator and stores the resulting elements for use with other split functions.

## Syntax

```zbr
ZtextSplit{text?;separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | No | - |
| separator | string | No | - |



## Example

```zbr
ZtextSplit{apple,banana,cherry;,}
```
