# ZsplitText

Returns a single element from the stored split text by 1-based index, or use < for the first or > for the last element.

## Syntax

```zbr
ZsplitText{index}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | string | Yes | - |



## Example

```zbr
ZsplitText{2}
```
