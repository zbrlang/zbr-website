# ZjoinSplitText

Joins all elements in the current split text into a single string using the given separator.

## Syntax

```zbr
ZjoinSplitText{separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | - |



## Example

```zbr
ZjoinSplitText{, }
```
