# ZgetTextSplitIndex

Returns the 1-based index of the first element in the split that matches the given value, or -1 if not found.

## Syntax

```zbr
ZgetTextSplitIndex{value?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| value | string | No | - |



## Example

```zbr
ZgetTextSplitIndex{banana}
```
