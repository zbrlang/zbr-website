# ZinviteInfo

Returns a specific field of a Discord invite by its code, where type is one of uses, channel, creationDate, inviter, or isTemporary.

## Syntax

```zbr
ZinviteInfo{code;type}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| code | string | Yes | - |
| type | string | Yes | - |



## Example

```zbr
ZinviteInfo{abc123;channel}
```
