# ZserverInvite

Returns the URL of the first available invite for the guild, or an empty string if none exist.

## Syntax

```zbr
ZserverInvite{guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZserverInvite{}
```
