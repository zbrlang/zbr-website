# ZchannelInvites

Returns a space-separated list of invite codes for the given channel.

## Syntax

```zbr
ZchannelInvites{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZchannelInvites{1234567890123456789}
```
