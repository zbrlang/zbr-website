# Zeval

Parses and executes a ZBR code string at runtime, returning the concatenated text output.

## Syntax

```zbr
Zeval{code?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| code | string | No | - |



## Example

```zbr
Zeval{Zhello{} ZloopIndex{}}
```
