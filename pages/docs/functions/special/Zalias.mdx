# Zalias

Creates a runtime alias for a ZBR function. The alias can be called using Z+alias_name+{} syntax.

## Syntax

```zbr
Zalias{function;alias_name}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| function | function | Yes | - |
| alias_name | string | Yes | - |



## Example

```zbr
Zalias{Zusername;greet} Zgreet{}
```
