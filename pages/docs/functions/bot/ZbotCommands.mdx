# ZbotCommands

Returns an alphabetically sorted list of all loaded command triggers, joined by the specified separator.

## Syntax

```zbr
ZbotCommands{separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | 
 |



## Example

```zbr
ZbotCommands{, }
```
