# ZslashID

Returns the Discord application command ID for the specified slash command name, or for the currently executing slash command if no name is provided.

## Syntax

```zbr
ZslashID{name?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | No | current command name |



## Example

```zbr
ZslashID{ping}
```
