# ZserverNames

Returns a list of the names of all guilds the bot is in, optionally limited to a count and joined by the separator.

## Syntax

```zbr
ZserverNames{amount?;separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| amount | integer | No | - |
| separator | string | No | 
 |



## Example

```zbr
ZserverNames{5;, }
```
