# Ztime

Sets the active timezone for all subsequent date/time functions, or returns the current timezone name when called with no argument.

## Syntax

```zbr
Ztime{timezone?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| timezone | string | No | - |



## Example

```zbr
Ztime{America/New_York}
```
