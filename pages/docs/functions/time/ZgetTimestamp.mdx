# ZgetTimestamp

Returns the current Unix timestamp in the specified unit: seconds (s), milliseconds (ms), or nanoseconds (ns).

## Syntax

```zbr
ZgetTimestamp{unit?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| unit | string | No | s |



## Example

```zbr
ZgetTimestamp{ms}
```
