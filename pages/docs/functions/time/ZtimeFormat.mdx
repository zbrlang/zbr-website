# ZtimeFormat

Executes the ZtimeFormat function.

## Syntax

```zbr
ZtimeFormat{arg1;arg2;arg3?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| arg1 | string | Yes | - |
| arg2 | string | Yes | - |
| arg3 | string | No | - |



## Example

```zbr
ZtimeFormat{}
```
