# ZpollAllowMultiselect

Allows voting for multiple answers in the poll being created.

## Syntax

```zbr
ZpollAllowMultiselect{true/false}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| value | boolean | Yes | - |



## Example

```zbr
ZpollAllowMultiselect{true}
```
