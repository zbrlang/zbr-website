# ZpollAddAnswer

Adds an answer option to the poll being created.

## Syntax

```zbr
ZpollAddAnswer{text;emoji?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| emoji | emoji | No | - |



## Example

```zbr
ZpollAddAnswer{Yes;✅}
```
