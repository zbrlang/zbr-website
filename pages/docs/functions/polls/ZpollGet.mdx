# ZpollGet

Gets poll details and current results for a message.

## Syntax

```zbr
ZpollGet{messageID;channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| messageID | snowflake | Yes | - |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZpollGet{987654321098765432}
```
