# ZpollCreate

Starts creating a poll with the given question and duration in hours.

## Syntax

```zbr
ZpollCreate{question;duration}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| question | string | Yes | - |
| duration | integer | Yes | - |



## Example

```zbr
ZpollCreate{What is your favorite color?;24}
```
