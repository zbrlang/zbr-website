# ZisMentioned

Returns true if the specified user is mentioned in the triggering message, defaulting to the command author.

## Syntax

```zbr
ZisMentioned{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | - |



## Example

```zbr
ZisMentioned{123456789012345678}
```
