# Zdm

Sends a direct message with the specified content to the given user.

## Syntax

```zbr
Zdm{userID;content}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | Yes | - |
| content | string | Yes | - |



## Example

```zbr
Zdm{123456789012345678;Hello, this is a private message!}
```
