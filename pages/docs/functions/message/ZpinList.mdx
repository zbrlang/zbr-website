# ZpinList

Returns a space-separated list of pinned message IDs in a channel.

## Syntax

```zbr
ZpinList{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | current channel |



## Example

```zbr
ZpinList{123456789012345678}
```
