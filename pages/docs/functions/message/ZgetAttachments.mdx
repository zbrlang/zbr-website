# ZgetAttachments

Returns attachment URLs from the triggering message joined by newlines, or the single URL at the specified 1-based index.

## Syntax

```zbr
ZgetAttachments{index?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | No | - |



## Example

```zbr
ZgetAttachments{1}
```
