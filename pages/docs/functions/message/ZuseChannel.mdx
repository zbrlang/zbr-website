# ZuseChannel

Redirects all bot output for the current command execution to the specified channel.

## Syntax

```zbr
ZuseChannel{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZuseChannel{123456789012345678}
```
