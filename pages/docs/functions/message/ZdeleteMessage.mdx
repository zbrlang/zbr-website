# ZdeleteMessage

Deletes the specified message from a channel.

## Syntax

```zbr
ZdeleteMessage{channelID;messageID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |
| messageID | snowflake | Yes | - |



## Example

```zbr
ZdeleteMessage{123456789012345678;987654321098765432}
```
