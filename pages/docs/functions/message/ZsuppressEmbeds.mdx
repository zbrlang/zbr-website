# ZsuppressEmbeds

Suppresses or un-suppresses link embeds on an existing message.

## Syntax

```zbr
ZsuppressEmbeds{messageID;suppress}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| messageID | snowflake | Yes | - |
| suppress | boolean | Yes | - |



## Example

```zbr
ZsuppressEmbeds{123456789012345678;true}
```
