# ZpublishMessage

Crossposts a message from an announcement channel to all subscribed channels.

## Syntax

```zbr
ZpublishMessage{channelID?;messageID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | - |
| messageID | snowflake | Yes | - |



## Example

```zbr
ZpublishMessage{123456789012345678;987654321098765432}
```
