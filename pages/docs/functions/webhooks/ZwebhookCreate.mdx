# ZwebhookCreate

Creates a webhook in the specified channel and returns its full URL.

## Syntax

```zbr
ZwebhookCreate{channelID;name}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |
| name | string | Yes | - |



## Example

```zbr
ZwebhookCreate{123456789012345678;Announcements Hook}
```
