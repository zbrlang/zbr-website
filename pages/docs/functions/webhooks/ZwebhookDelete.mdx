# ZwebhookDelete

Deletes a webhook identified by its full Discord webhook URL.

## Syntax

```zbr
ZwebhookDelete{webhookURL}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| webhookURL | url | Yes | - |



## Example

```zbr
ZwebhookDelete{https://discord.com/api/webhooks/123456789/abcdeftoken}
```
