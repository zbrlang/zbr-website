# Zpad

Executes the Zpad function.

## Syntax

```zbr
Zpad{arg1;arg2;arg3?;arg4?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| arg1 | string | Yes | - |
| arg2 | string | Yes | - |
| arg3 | string | No | - |
| arg4 | string | No | - |



## Example

```zbr
Zpad{}
```
