# ZindexOf

Returns the 1-based character position of the first occurrence of a search string in the text, or 0 if not found.

## Syntax

```zbr
ZindexOf{text;search}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| search | string | Yes | - |



## Example

```zbr
ZindexOf{Hello World;World}
```
