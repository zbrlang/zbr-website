# Ztitlecase

Converts the first letter of each word to uppercase and all remaining letters to lowercase.

## Syntax

```zbr
Ztitlecase{text}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |



## Example

```zbr
Ztitlecase{hello world}
```
