# ZtrimContent

Collapses multiple consecutive spaces into a single space and trims leading and trailing whitespace.

## Syntax

```zbr
ZtrimContent{text}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |



## Example

```zbr
ZtrimContent{hello   world}
```
