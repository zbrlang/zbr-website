# ZpadLeft

Pads the text on the left with a repeated character until it reaches the specified width, defaulting to a space.

## Syntax

```zbr
ZpadLeft{text;width;char?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| width | integer | Yes | - |
| char | string | No |   |



## Example

```zbr
ZpadLeft{42;5;0}
```
