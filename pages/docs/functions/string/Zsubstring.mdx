# Zsubstring

Returns a portion of the text starting at a 1-based character position, optionally limited to a specified length.

## Syntax

```zbr
Zsubstring{text;start;length?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| start | integer | Yes | - |
| length | integer | No | - |



## Example

```zbr
Zsubstring{Hello World;7;5}
```
