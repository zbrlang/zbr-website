# ZstartsWith

Returns true if the text begins with the specified prefix.

## Syntax

```zbr
ZstartsWith{text;prefix}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| prefix | string | Yes | - |



## Example

```zbr
ZstartsWith{Hello World;Hello}
```
