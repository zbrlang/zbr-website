# ZendsWith

Returns true if the text ends with the specified suffix.

## Syntax

```zbr
ZendsWith{text;suffix}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| suffix | string | Yes | - |



## Example

```zbr
ZendsWith{Hello World;World}
```
