# ZlinesCount

Returns the number of lines in the text, returning zero for an empty string.

## Syntax

```zbr
ZlinesCount{text}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |



## Example

```zbr
ZlinesCount{line one
line two}
```
