# ZcheckContains

Returns true if the text contains any of the provided words, using a case-insensitive comparison.

## Syntax

```zbr
ZcheckContains{text;word;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| text | string | Yes | - |
| word | string | Yes | - |



## Example

```zbr
ZcheckContains{Hello World;hello;bye}
```
