# Zurl

Encodes or decodes a string using URL percent-encoding based on the specified mode.

## Syntax

```zbr
Zurl{mode;text}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| mode | string | Yes | - |
| text | string | Yes | - |



## Example

```zbr
Zurl{encode;hello world}
```
