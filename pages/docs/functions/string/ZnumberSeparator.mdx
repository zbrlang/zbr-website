# ZnumberSeparator

Formats a number string by inserting thousands separators, defaulting to a comma.

## Syntax

```zbr
ZnumberSeparator{number;separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| number | string | Yes | - |
| separator | string | No | , |



## Example

```zbr
ZnumberSeparator{1000000}
```
