# ZjsonArrayReverse

Reverses the array at the specified key path in the working JSON object in place.

## Syntax

```zbr
ZjsonArrayReverse{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonArrayReverse{scores}
```
