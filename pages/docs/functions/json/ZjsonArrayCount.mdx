# ZjsonArrayCount

Returns the number of elements in the array at the specified key path in the working JSON object.

## Syntax

```zbr
ZjsonArrayCount{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonArrayCount{users}
```
