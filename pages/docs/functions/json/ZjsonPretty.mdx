# ZjsonPretty

Serializes the working JSON object to a pretty-printed JSON string using the specified number of spaces for indentation.

## Syntax

```zbr
ZjsonPretty{indent?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| indent | integer | No | 2 |



## Example

```zbr
ZjsonPretty{4}
```
