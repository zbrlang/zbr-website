# ZjsonArrayUnshift

Inserts a value at the beginning of the array at the specified key path in the working JSON object.

## Syntax

```zbr
ZjsonArrayUnshift{key;...;value}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |
| value | string | Yes | - |



## Example

```zbr
ZjsonArrayUnshift{users;Alice}
```
