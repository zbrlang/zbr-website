# ZjsonArrayAppend

Appends a value to the end of the array at the specified key path in the working JSON object.

## Syntax

```zbr
ZjsonArrayAppend{key;...;value}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |
| value | string | Yes | - |



## Example

```zbr
ZjsonArrayAppend{users;Alice}
```
