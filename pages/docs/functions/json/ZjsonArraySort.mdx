# ZjsonArraySort

Sorts the array at the specified key path in ascending order, comparing numbers numerically and other values lexicographically.

## Syntax

```zbr
ZjsonArraySort{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonArraySort{scores}
```
