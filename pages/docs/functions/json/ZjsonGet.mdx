# ZjsonGet

Returns the value at the specified key path in the working JSON object, returning an empty string if the path does not exist.

## Syntax

```zbr
ZjsonGet{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonGet{user;name}
```
