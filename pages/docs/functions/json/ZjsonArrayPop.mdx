# ZjsonArrayPop

Removes and returns the last element of the array at the specified key path in the working JSON object.

## Syntax

```zbr
ZjsonArrayPop{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonArrayPop{users}
```
