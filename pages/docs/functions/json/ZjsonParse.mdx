# ZjsonParse

Parses a JSON string and stores it as the working JSON object for the current execution.

## Syntax

```zbr
ZjsonParse{jsonString}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| jsonString | string | Yes | - |



## Example

```zbr
ZjsonParse{{"name":"Alice","age":30}}
```
