# ZjsonExists

Returns true if the specified key path exists in the working JSON object, or false otherwise.

## Syntax

```zbr
ZjsonExists{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonExists{user;email}
```
