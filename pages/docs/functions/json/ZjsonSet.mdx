# ZjsonSet

Sets a value at the specified key path in the working JSON object, automatically inferring the value type.

## Syntax

```zbr
ZjsonSet{key;...;value}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |
| value | string | Yes | - |



## Example

```zbr
ZjsonSet{user;name;Alice}
```
