# ZjsonArrayShift

Removes and returns the first element of the array at the specified key path in the working JSON object.

## Syntax

```zbr
ZjsonArrayShift{key;...}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| key | string | Yes | - |



## Example

```zbr
ZjsonArrayShift{users}
```
