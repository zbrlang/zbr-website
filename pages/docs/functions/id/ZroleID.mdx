# ZroleID

Returns the command author's top role ID, or searches for and returns the ID of a guild role matching the given name.

## Syntax

```zbr
ZroleID{name?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | No | author's top role ID |



## Example

```zbr
ZroleID{Moderator}
```
