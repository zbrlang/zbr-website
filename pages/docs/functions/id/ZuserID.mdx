# ZuserID

Returns the command author's user ID, or searches for and returns the user ID of a guild member matching the given name.

## Syntax

```zbr
ZuserID{name?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | No | author's ID |



## Example

```zbr
ZuserID{JohnDoe}
```
