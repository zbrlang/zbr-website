# ZchannelID

Returns the current channel's ID, or searches for and returns the ID of a guild channel matching the given name.

## Syntax

```zbr
ZchannelID{name?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | No | current channel's ID |



## Example

```zbr
ZchannelID{general}
```
