# ZdeleteSticker

Deletes a guild sticker from the current server by its sticker ID.

## Syntax

```zbr
ZdeleteSticker{stickerID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| stickerID | snowflake | Yes | - |



## Example

```zbr
ZdeleteSticker{1234567890123456789}
```
