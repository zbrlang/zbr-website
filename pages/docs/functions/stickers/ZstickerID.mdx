# ZstickerID

Returns the ID of a guild sticker matching the given name, or an empty string if not found.

## Syntax

```zbr
ZstickerID{name}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |



## Example

```zbr
ZstickerID{CoolSticker}
```
