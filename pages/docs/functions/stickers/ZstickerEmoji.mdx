# ZstickerEmoji

Returns the related emoji tag for a guild sticker as a string.

## Syntax

```zbr
ZstickerEmoji{stickerID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| stickerID | snowflake | Yes | - |



## Example

```zbr
ZstickerEmoji{1234567890123456789}
```
