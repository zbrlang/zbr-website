# ZstickerExists

Returns true if a sticker with the given ID exists in the current server, or false otherwise.

## Syntax

```zbr
ZstickerExists{stickerID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| stickerID | snowflake | Yes | - |



## Example

```zbr
ZstickerExists{1234567890123456789}
```
