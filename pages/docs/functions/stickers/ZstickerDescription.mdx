# ZstickerDescription

Returns the description of a guild sticker by its sticker ID, or an empty string if no description is set.

## Syntax

```zbr
ZstickerDescription{stickerID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| stickerID | snowflake | Yes | - |



## Example

```zbr
ZstickerDescription{1234567890123456789}
```
