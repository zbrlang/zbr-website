# ZstageEdit

Edits the topic of an existing stage instance in the specified stage channel.

## Syntax

```zbr
ZstageEdit{channelID;topic}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |
| topic | string | Yes | - |



## Example

```zbr
ZstageEdit{987654321012345678;New Topic Here}
```
