# ZstageSpeakers

Returns a comma-separated list of user IDs who are currently speaking in a stage channel.

## Syntax

```zbr
ZstageSpeakers{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZstageSpeakers{123456789012345678}
```
