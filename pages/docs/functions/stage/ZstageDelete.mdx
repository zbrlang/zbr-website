# ZstageDelete

Ends and deletes the stage instance associated with the specified stage channel.

## Syntax

```zbr
ZstageDelete{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZstageDelete{987654321012345678}
```
