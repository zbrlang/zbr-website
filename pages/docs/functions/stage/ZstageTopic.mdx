# ZstageTopic

Returns the current topic of the stage instance in the specified stage channel.

## Syntax

```zbr
ZstageTopic{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZstageTopic{987654321012345678}
```
