# ZstageAudience

Returns a comma-separated list of user IDs who are in the audience of a stage channel.

## Syntax

```zbr
ZstageAudience{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZstageAudience{123456789012345678}
```
