# ZhttpDelete

Sends an HTTP DELETE request to the specified URL with an optional request body and stores the response.

## Syntax

```zbr
ZhttpDelete{url;body?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| body | string | No | - |



## Example

```zbr
ZhttpDelete{https://api.example.com/users/1}
```
