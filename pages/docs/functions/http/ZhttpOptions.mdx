# ZhttpOptions

Sends an HTTP OPTIONS request to the specified URL and stores the response for access via ZhttpStatus and ZhttpResult.

## Syntax

```zbr
ZhttpOptions{url}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |



## Example

```zbr
ZhttpOptions{https://api.example.com/users/1}
```
