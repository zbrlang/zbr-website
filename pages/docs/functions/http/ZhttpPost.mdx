# ZhttpPost

Sends an HTTP POST request to the specified URL with an optional request body and stores the response.

## Syntax

```zbr
ZhttpPost{url;body?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| url | url | Yes | - |
| body | string | No | - |



## Example

```zbr
ZhttpPost{https://api.example.com/users;{"name":"Alice"}}
```
