# Zrepeat

Executes the given code block N times and returns each non-empty output joined by newlines.

## Syntax

```zbr
Zrepeat{N;code}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| N | integer | Yes | - |
| code | string | Yes | - |



## Example

```zbr
Zrepeat{3;Line ZloopIndex{}}
```
