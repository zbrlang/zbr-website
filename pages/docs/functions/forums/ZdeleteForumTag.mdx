# ZdeleteForumTag

Removes an available tag from a forum channel by its tag ID.

## Syntax

```zbr
ZdeleteForumTag{channelID;tagID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |
| tagID | snowflake | Yes | - |



## Example

```zbr
ZdeleteForumTag{987654321012345678;1122334455667788}
```
