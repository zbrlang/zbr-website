# ZpostTags

Returns a space-separated list of tag IDs currently applied to a forum post thread.

## Syntax

```zbr
ZpostTags{threadID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| threadID | snowflake | Yes | - |



## Example

```zbr
ZpostTags{1234567890123456789}
```
