# ZforumPostCount

Returns the number of currently active posts in a forum channel.

## Syntax

```zbr
ZforumPostCount{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZforumPostCount{987654321012345678}
```
