# ZforumPosts

Returns a space-separated list of active post IDs in a forum channel.

## Syntax

```zbr
ZforumPosts{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZforumPosts{987654321012345678}
```
