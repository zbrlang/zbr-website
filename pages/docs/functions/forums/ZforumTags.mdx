# ZforumTags

Returns a space-separated list of available tag names in a forum channel.

## Syntax

```zbr
ZforumTags{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZforumTags{987654321012345678}
```
