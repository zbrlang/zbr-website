# ZforumTagEmoji

Returns the emoji associated with a forum tag.

## Syntax

```zbr
ZforumTagEmoji{channelID;tagID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |
| tagID | snowflake | Yes | - |



## Example

```zbr
ZforumTagEmoji{987654321012345678;1122334455667788}
```
