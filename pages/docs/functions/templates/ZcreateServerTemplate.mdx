# ZcreateServerTemplate

Creates a new server template from the current guild's structure. Returns the template code.

## Syntax

```zbr
ZcreateServerTemplate{name;description?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |
| description | string | No | - |



## Example

```zbr
ZcreateServerTemplate{My Template;Cool server layout}
```
