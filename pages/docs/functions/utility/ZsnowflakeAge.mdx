# ZsnowflakeAge

Returns a human-readable age from any snowflake like '3 days ago'.

## Syntax

```zbr
ZsnowflakeAge{snowflakeID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| snowflakeID | string | Yes | - |



## Example

```zbr
ZsnowflakeAge{123456789012345678}
```
