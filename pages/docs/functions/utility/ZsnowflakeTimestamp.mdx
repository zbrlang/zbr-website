# ZsnowflakeTimestamp

Extracts the Unix timestamp from any Discord snowflake ID.

## Syntax

```zbr
ZsnowflakeTimestamp{snowflakeID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| snowflakeID | string | Yes | - |



## Example

```zbr
ZsnowflakeTimestamp{123456789012345678}
```
