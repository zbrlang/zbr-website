# ZuserBadge

Returns the public profile badges of the command author or specified user as a separated list.

## Syntax

```zbr
ZuserBadge{userID?;separator?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |
| separator | string | No | 
 |



## Example

```zbr
ZuserBadge{123456789012345678;, }
```
