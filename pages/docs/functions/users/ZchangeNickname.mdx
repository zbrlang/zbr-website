# ZchangeNickname

Changes the guild nickname of the command author or specified user; supports the %username% placeholder in the nickname string.

## Syntax

```zbr
ZchangeNickname{nickname;userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| nickname | string | Yes | - |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZchangeNickname{CoolNick;123456789012345678}
```
