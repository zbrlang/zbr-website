# ZisAdmin

Returns true if the command author or specified user has the Administrator permission in the current guild, otherwise false.

## Syntax

```zbr
ZisAdmin{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZisAdmin{123456789012345678}
```
