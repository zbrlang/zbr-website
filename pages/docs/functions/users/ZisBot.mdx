# ZisBot

Returns true if the command author or specified user is a bot account, otherwise false.

## Syntax

```zbr
ZisBot{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZisBot{123456789012345678}
```
