# ZuserBanner

Returns the banner URL of the command author or specified user, or an empty string if no banner is set.

## Syntax

```zbr
ZuserBanner{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZuserBanner{123456789012345678}
```
