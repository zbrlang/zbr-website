# ZisModerator

Returns true if the command author or specified user has the Moderate Members permission in the current guild, otherwise false.

## Syntax

```zbr
ZisModerator{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZisModerator{123456789012345678}
```
