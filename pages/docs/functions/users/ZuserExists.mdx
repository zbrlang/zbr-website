# ZuserExists

Returns true if the given user ID resolves to a valid Discord user, or false otherwise.

## Syntax

```zbr
ZuserExists{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZuserExists{123456789012345678}
```
