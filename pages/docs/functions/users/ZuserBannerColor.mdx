# ZuserBannerColor

Returns the banner accent color of the command author or specified user as a hex string, or an empty string if none is set.

## Syntax

```zbr
ZuserBannerColor{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZuserBannerColor{123456789012345678}
```
