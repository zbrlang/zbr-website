# ZuserAvatar

Returns the avatar URL of the command author, or of the specified user if a user ID is provided.

## Syntax

```zbr
ZuserAvatar{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZuserAvatar{123456789012345678}
```
