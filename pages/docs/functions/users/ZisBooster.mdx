# ZisBooster

Returns true if the command author or specified user is currently boosting the current guild, otherwise false.

## Syntax

```zbr
ZisBooster{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author's ID |



## Example

```zbr
ZisBooster{123456789012345678}
```
