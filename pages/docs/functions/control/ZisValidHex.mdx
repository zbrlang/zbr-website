# ZisValidHex

Returns true if the value is a valid six-digit hex color code, otherwise returns false.

## Syntax

```zbr
ZisValidHex{value}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| value | string | Yes | - |



## Example

```zbr
ZisValidHex{#FF5733}
```
