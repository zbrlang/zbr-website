# ZisTimedOut

Returns true if the specified user currently has an active communication timeout.

## Syntax

```zbr
ZisTimedOut{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZisTimedOut{123456789012345678}
```
