# Zdeafen

Server-deafens the specified user so they cannot hear any audio in voice channels.

## Syntax

```zbr
Zdeafen{userID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | Yes | - |



## Example

```zbr
Zdeafen{123456789012345678}
```
