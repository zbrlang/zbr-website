# Zkick

Kicks a user from the guild with an optional reason.

## Syntax

```zbr
Zkick{userID;reason?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | Yes | - |
| reason | string | No | - |



## Example

```zbr
Zkick{123456789012345678;Rule violation}
```
