# ZisBanned

Returns true if the specified user is currently banned from the guild.

## Syntax

```zbr
ZisBanned{userID;guildID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | Yes | - |
| guildID | snowflake | No | current guild |



## Example

```zbr
ZisBanned{123456789012345678}
```
