# Zundeafen

Removes the server deafen from the specified user so they can hear audio in voice channels again.

## Syntax

```zbr
Zundeafen{userID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | Yes | - |



## Example

```zbr
Zundeafen{123456789012345678}
```
