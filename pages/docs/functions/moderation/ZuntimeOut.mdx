# ZuntimeOut

Removes the active communication timeout from a user, restoring their ability to send messages.

## Syntax

```zbr
ZuntimeOut{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZuntimeOut{123456789012345678}
```
