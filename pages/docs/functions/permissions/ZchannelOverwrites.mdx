# ZchannelOverwrites

Returns the full permission overwrite list on a channel as a JSON array.

## Syntax

```zbr
ZchannelOverwrites{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZchannelOverwrites{123456789012345678}
```
