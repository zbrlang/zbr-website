# ZonlyAdmin

Halts execution unless the author has the Administrator permission in the guild.

## Syntax

```zbr
ZonlyAdmin{errorMessage?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| errorMessage | string | No | You need Administrator permission to use this command. |



## Example

```zbr
ZonlyAdmin{Admins only!}
```
