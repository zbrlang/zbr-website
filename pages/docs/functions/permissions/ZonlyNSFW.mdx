# ZonlyNSFW

Halts execution unless the current channel is marked as NSFW.

## Syntax

```zbr
ZonlyNSFW{errorMessage?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| errorMessage | string | No | This command can only be used in NSFW channels. |



## Example

```zbr
ZonlyNSFW{This command requires an NSFW channel.}
```
