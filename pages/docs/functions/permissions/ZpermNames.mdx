# ZpermNames

Converts a permission bitfield integer to a comma-separated list of human-readable permission names.

## Syntax

```zbr
ZpermNames{bitfield}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| bitfield | integer | Yes | - |



## Example

```zbr
ZpermNames{8}
```
