# ZgetRoleSelectRoleIDs

Returns all selected role IDs from a role select menu interaction joined by the given separator.

## Syntax

```zbr
ZgetRoleSelectRoleIDs{separator?;limit?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | ,  |
| limit | integer | No | - |



## Example

```zbr
ZgetRoleSelectRoleIDs{, ;2}
```
