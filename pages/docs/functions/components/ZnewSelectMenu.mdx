# ZnewSelectMenu

Creates a string select menu on the bot's response.

## Syntax

```zbr
ZnewSelectMenu{menuID;min?;max?;placeholder?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| menuID | string | Yes | - |
| min | integer | No | 1 |
| max | integer | No | 1 |
| placeholder | string | No | - |



## Example

```zbr
ZnewSelectMenu{color_menu;1;1;Pick a color}
```
