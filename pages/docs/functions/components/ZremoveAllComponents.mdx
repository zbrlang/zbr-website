# ZremoveAllComponents

Removes all components from a message, defaulting to the triggering message.

## Syntax

```zbr
ZremoveAllComponents{messageID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| messageID | snowflake | No | - |



## Example

```zbr
ZremoveAllComponents{1234567890123456789}
```
