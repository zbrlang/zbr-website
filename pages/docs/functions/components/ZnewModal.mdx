# ZnewModal

Creates a modal dialog to collect text input from the user.

## Syntax

```zbr
ZnewModal{modalID;title}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| modalID | string | Yes | - |
| title | string | Yes | - |



## Example

```zbr
ZnewModal{feedback_modal;Submit Feedback}
```
