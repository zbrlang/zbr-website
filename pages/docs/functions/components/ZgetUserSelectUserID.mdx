# ZgetUserSelectUserID

Returns the selected user ID at the given 1-based index from a user select menu interaction.

## Syntax

```zbr
ZgetUserSelectUserID{index}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | Yes | - |



## Example

```zbr
ZgetUserSelectUserID{1}
```
