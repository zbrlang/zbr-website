# ZgetMentionableSelectUserID

Returns the selected user or role ID at the given 1-based index from a mentionable select menu interaction.

## Syntax

```zbr
ZgetMentionableSelectUserID{index}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | Yes | - |



## Example

```zbr
ZgetMentionableSelectUserID{1}
```
