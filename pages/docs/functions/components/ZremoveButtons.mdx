# ZremoveButtons

Removes all buttons from a message by clearing all its components, defaulting to the triggering message.

## Syntax

```zbr
ZremoveButtons{messageID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| messageID | snowflake | No | - |



## Example

```zbr
ZremoveButtons{1234567890123456789}
```
