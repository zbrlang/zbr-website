# ZgetRoleSelectRoleID

Returns the selected role ID at the given 1-based index from a role select menu interaction.

## Syntax

```zbr
ZgetRoleSelectRoleID{index}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| index | integer | Yes | - |



## Example

```zbr
ZgetRoleSelectRoleID{1}
```
