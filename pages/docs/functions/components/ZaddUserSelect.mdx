# ZaddUserSelect

Adds a user select menu component to the bot's response.

## Syntax

```zbr
ZaddUserSelect{menuID;min?;max?;placeholder?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| menuID | string | Yes | - |
| min | integer | No | 1 |
| max | integer | No | 1 |
| placeholder | string | No | - |



## Example

```zbr
ZaddUserSelect{user_select;1;1;Pick a user}
```
