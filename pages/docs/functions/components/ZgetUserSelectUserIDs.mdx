# ZgetUserSelectUserIDs

Returns all selected user IDs from a user select menu interaction joined by the given separator.

## Syntax

```zbr
ZgetUserSelectUserIDs{separator?;limit?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| separator | string | No | ,  |
| limit | integer | No | - |



## Example

```zbr
ZgetUserSelectUserIDs{, ;3}
```
