# ZvoiceEmpty

Returns true if the specified voice channel has zero members; defaults to the author's current voice channel.

## Syntax

```zbr
ZvoiceEmpty{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | author's voice channel |



## Example

```zbr
ZvoiceEmpty{123456789012345678}
```
