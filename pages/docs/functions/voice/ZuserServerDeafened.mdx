# ZuserServerDeafened

Returns true if the specified user has been server-deafened by a moderator, defaulting to the author.

## Syntax

```zbr
ZuserServerDeafened{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZuserServerDeafened{123456789012345678}
```
