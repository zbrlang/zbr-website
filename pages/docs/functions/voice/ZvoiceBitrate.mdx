# ZvoiceBitrate

Returns the bitrate in kbps of the specified voice channel, defaulting to the author's current voice channel.

## Syntax

```zbr
ZvoiceBitrate{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | author's voice channel |



## Example

```zbr
ZvoiceBitrate{123456789012345678}
```
