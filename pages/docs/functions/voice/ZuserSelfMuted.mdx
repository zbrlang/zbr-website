# ZuserSelfMuted

Returns true if the specified user has muted themselves in voice, defaulting to the author.

## Syntax

```zbr
ZuserSelfMuted{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZuserSelfMuted{123456789012345678}
```
