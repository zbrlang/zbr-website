# ZvoiceQuality

Returns the video quality mode of a voice channel (AUTO or FULL).

## Syntax

```zbr
ZvoiceQuality{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZvoiceQuality{123456789012345678}
```
