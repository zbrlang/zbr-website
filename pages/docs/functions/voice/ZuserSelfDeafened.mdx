# ZuserSelfDeafened

Returns true if the specified user has deafened themselves in voice, defaulting to the author.

## Syntax

```zbr
ZuserSelfDeafened{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZuserSelfDeafened{123456789012345678}
```
