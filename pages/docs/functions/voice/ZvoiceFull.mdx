# ZvoiceFull

Returns true if the specified voice channel's member count equals its user limit; defaults to the author's current voice channel.

## Syntax

```zbr
ZvoiceFull{channelID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | No | author's voice channel |



## Example

```zbr
ZvoiceFull{123456789012345678}
```
