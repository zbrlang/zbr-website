# ZisInVoice

Returns true if the specified user is currently connected to a voice channel, defaulting to the author.

## Syntax

```zbr
ZisInVoice{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZisInVoice{123456789012345678}
```
