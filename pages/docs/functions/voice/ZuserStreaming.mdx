# ZuserStreaming

Returns true if the specified user is currently streaming in a voice channel, defaulting to the author.

## Syntax

```zbr
ZuserStreaming{userID?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| userID | snowflake | No | author |



## Example

```zbr
ZuserStreaming{123456789012345678}
```
