# ZvoiceRegion

Returns the voice region of a voice channel (e.g., 'us-west', 'automatic').

## Syntax

```zbr
ZvoiceRegion{channelID}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| channelID | snowflake | Yes | - |



## Example

```zbr
ZvoiceRegion{123456789012345678}
```
