# ZsetVar

Sets the value of a global bot-wide variable by name.

## Syntax

```zbr
ZsetVar{name;value?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |
| value | string | No | - |



## Example

```zbr
ZsetVar{maintenance_mode;true}
```
