# Zvar

Gets or sets a temporary in-memory variable that persists only for the duration of the current command execution.

## Syntax

```zbr
Zvar{name;value?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| name | string | Yes | - |
| value | string | No | - |



## Example

```zbr
Zvar{counter;0}
```
