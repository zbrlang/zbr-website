# Zerror

Halts execution and displays a custom user-facing error message.

## Syntax

```zbr
Zerror{message}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| message | string | No | - |



## Example

```zbr
Zerror{You do not have permission to use this command.}
```
