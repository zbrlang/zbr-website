# ZchangeCooldownTime

Overrides the label words used when formatting remaining cooldown times.

## Syntax

```zbr
ZchangeCooldownTime{days?;hours?;minutes?;seconds?}
```


## Arguments

| Name | Type | Required | Default |
| :--- | :--- | :--- | :--- |
| days | string | No | Days |
| hours | string | No | Hours |
| minutes | string | No | Minutes |
| seconds | string | No | Seconds |



## Example

```zbr
ZchangeCooldownTime{días;horas;minutos;segundos}
```
