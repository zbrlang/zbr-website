# onReactionAdd

Fires when a reaction is added to a message.

## When it fires

Reaction is added


## Available Context Variables

| Variable | Description |
| :--- | :--- |
| author_id | User ID who reacted |
| username | Empty |
| channel_id | Channel ID |
| guild_id | Guild ID (if applicable) |
| message | Empty |
| trigger_message_id | Message ID that was reacted to |
