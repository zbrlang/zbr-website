# onReactionRemove

Fires when a reaction is removed from a message.

## When it fires

Reaction is removed


## Available Context Variables

| Variable | Description |
| :--- | :--- |
| author_id | User ID who removed reaction |
| username | Empty |
| channel_id | Channel ID |
| guild_id | Guild ID (if applicable) |
| message | Empty |
| trigger_message_id | Message ID that had reaction removed |
